# Studio Ghibli To Do List

A **To Do List web application** inspired by **Studio Ghibli** aesthetics.  
This project allows you to add, complete, and remove tasks, and keeps track of your daily progress with a progress bar.

---

## Features

- Add new tasks dynamically
- Mark tasks as completed
- Delete tasks
- “Finish Day” button to toggle all tasks
- Progress bar showing percentage of tasks completed
- Local storage saves your tasks between sessions
- Date display for day and month
- Studio Ghibli themed design with custom images

---

## Technologies Used

- **HTML5** – structure of the application
- **CSS3** – custom styling and Ghibli theme
- **JavaScript** – functionality and local storage

## How to Use

1. Open `index.html` in a browser. OR use this link `https://hnrlina.github.io/ghiblist/`
2. Type a task in the input box and click the **+** button.
3. Click on a task to mark it as completed.
4. Click the **×** button on a task to remove it.
5. Click **FINISH DAY** to mark all tasks as completed or reset them.
6. Track your progress with the progress bar.
